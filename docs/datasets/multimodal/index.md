# Multimodal Datasets

Visual document benchmarks for image-based retrieval.

## Available Datasets

| Dataset | Description |
|---------|-------------|
| [ViDoRe](vidore.md) | Visual Document Retrieval |
| [ViDoRe v2](vidorev2.md) | Visual Document Retrieval v2 |
| [ViDoRe v3](vidorev3.md) | Visual Document Retrieval v3 |
| [VisRAG](visrag.md) | Visual RAG benchmark |

## Key Characteristics

- Documents are stored as images (PDF pages, screenshots)
- Requires multimodal embedding models (ColPali)
- BM25 text search not available
- GPU recommended for embedding generation
